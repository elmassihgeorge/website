Mathematical Foundation

The Mandelbrot set is defined as the set of complex numbers c for which the iterative formula zn+1 = zn2 + c does not diverge when starting from z0 = 0.

function calculateMandelbrot(x, y) { let a = map(x, 0, width, -2.5/zoom + offsetX, 1.5/zoom + offsetX); let b = map(y, 0, height, -1.5/zoom + offsetY, 1.5/zoom + offsetY); let ca = a, cb = b, n = 0, za = a, zb = b; while (n < maxIterations) { let aa = za * za - zb * zb; let bb = 2 * za * zb; za = aa + ca; zb = bb + cb; if (za * za + zb * zb > 4) { // Smooth coloring using continuous iteration count return n + 1 - log(log(sqrt(za*za + zb*zb)))/log(2); } n++; } return maxIterations; }

Escape-Time Algorithm

The visualization uses an escape-time algorithm to determine if each point belongs to the set. Points that don't escape after a maximum number of iterations are considered part of the set and colored black.

The algorithm's efficiency is crucial for real-time interaction. Several optimizations are employed:

  • Adaptive quality rendering during user interaction
  • Smooth iteration counting for better color gradients
  • Optimized pixel manipulation for performance
  • Dynamic iteration count based on zoom level

Color Mapping

Different color palettes highlight various aspects of the fractal's structure. The coloring is based on the number of iterations required for a point to escape the bounded region.

function getColor(iterations) { if (iterations >= maxIterations) return color(0, 0, 0); let palette = palettes[colorPalette]; let hue = map(iterations % 80, 0, 80, palette.hueRange[0], palette.hueRange[1]); let sat = palette.saturation; let bright = map(iterations, 0, maxIterations, 20, 100); return color(hue, sat, bright); }
